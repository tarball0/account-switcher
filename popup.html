How to Use:

  • Open any Google app (Gmail, Calendar, Keep, etc.) and press Alt + number to switch to a particular account.
  • For example, to switch between Gmail accounts, try pressing Alt + 1, Alt + 2, etc.
  • Almost all Google apps are supported. Send a request if an app you need is not covered.
  • For support or to report an issue, please send an email.
  • Source code for the extension is available on GitHub.
  • Disclaimer: This extension is not affiliated with Google in any way.
