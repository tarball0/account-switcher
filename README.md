# `account-switcher`
Chrome Extension for Switching Google Account in Web Apps.  
Forked from [here](https://github.com/anshulahuja98/account-switcher/)


### What does this extension do ?
This extension is for people using multiple Google accounts in their browser.
It often becomes a hassle switching between the different accounts for various Google apps like Gmail, Google Docs, Google Drive etc.
This extension gives you the ability to switch accounts seamlessly using keyboard shortcuts in firefox. 

Google gives the logged in accounts numbers, like: 0, 1, 2...
![image](https://user-images.githubusercontent.com/36476228/79776588-9b584900-8353-11ea-9ddb-ec6aa25fa018.png)

This extension makes use of that numbering.

## Usage
Press `Ctrl + <num>` to switch to the account associated with the specified number in the current page.  
Press `Ctrl + Shift + <num>` to open the account in a new tab.
